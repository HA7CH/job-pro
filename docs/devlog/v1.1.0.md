## 新增功能

### `--scope social|campus|intern|all`（全 50 家适配器）

**核心接口变更：** `cli/src/adapter.ts` 新增 `PositionScope` 类型 + `supportedScopes` 字段约定。

```ts
// adapter.ts
export type PositionScope = "social" | "campus" | "intern" | "all";
```

每个 adapter 声明自己支持哪些 scope：

```ts
export const supportedScopes = ["social", "campus", "intern", "all"] as const;
```

dispatcher（`index.ts`）在调用前检查 scope 是否在 `supportedScopes` 里，不支持则 fail-fast 返回明确错误，而不是静默返回空结果。

**修改的文件（60 个）：** 几乎所有 adapter + `index.ts` + `smoke.ts`

主要 scope 映射方式：

| ATS 家族 | campus 映射 | intern 映射 | social 映射 |
|---|---|---|---|
| ByteDance/ATSX | `recruitment_id_list:["201"]` | `recruitment_id_list:["202"]` | 无 filter |
| Feishu | `recruitment_id_list:["1"]` (校招) | `recruitment_id_list:["2"]` (实习) | `recruitment_id_list:["101"]` |
| Moka | `recruitType:"campus"` channel | `recruitType:"intern"` channel | `recruitType:"social"` channel |
| Beisen Wecruit | `recruitment_type` query param | 同上 | 同上 |
| Bespoke | 各公司独立实现 | 各公司独立实现 | 各公司独立实现 |

**新增设计文档：** `docs/social-hire-design.md`（406 行）— 记录各 ATS 家族的 scope 实现细节和探针结果。
