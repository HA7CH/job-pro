# v1.1.0 更新说明

> 发布日期：2026-05-20

---

## 🎉 这次更新做了什么？

**重大新功能：搜索时可以指定招聘类型了。**

之前搜索岗位只能拿到默认的校招结果，现在可以明确指定你想找的是校招、社招还是实习岗位。

---

## ✨ 新增功能

### `--scope` 参数：按招聘类型筛选

在任何搜索命令后面加上 `--scope`，可以指定岗位类型：

```bash
# 只看校招岗位（应届生）
job-pro bytedance search "算法" --scope campus

# 只看实习岗位
job-pro bytedance search "算法" --scope intern

# 只看社招岗位（有工作经验）
job-pro bytedance search "算法" --scope social

# 全部类型
job-pro bytedance search "算法" --scope all
```

**覆盖全部 50 家公司**，包括字节跳动、阿里巴巴、百度、腾讯等。

---

## 📋 各公司支持情况说明

不同公司对不同招聘类型的支持程度不同：

- 大多数公司同时支持校招、实习、社招
- 部分公司（如零一万物）只有社招，没有校招渠道
- 搜索不支持的类型时，系统会提示"该公司不支持此招聘类型"，而不是返回错误或空结果
