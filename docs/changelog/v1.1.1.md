# v1.1.1 更新说明

> 发布日期：2026-05-21

---

## 🐛 修复的问题

### 京东搜索结果为空

**之前的表现：** `jd search` 或 `jd match` 经常返回空结果，即使确实有在招岗位。

**现在的表现：** 京东搜索现在同时扫描应届、实习、人才三个招聘池，结果更完整。使用 `--scope intern` 只看实习，`--scope campus` 只看应届。

---

### 滴滴搜索丢失岗位

**之前的表现：** 使用 `--scope campus` 或 `--scope social` 筛选时，几乎所有岗位都被过滤掉了。

**现在的表现：** 已修复岗位 ID 格式识别问题，正常返回筛选后的岗位。

---

### 微博默认显示错误

**之前的表现：** 直接运行 `weibo search` 没有结果（微博校招渠道已关闭）。

**现在的表现：** 默认切换为社招渠道，直接运行 `weibo search` 可正常获取结果。
