# v1.1.2 更新说明

> 发布日期：2026-05-21

---

## 🐛 修复的问题

### 平安搜索报错（使用 `--scope` 时）

**之前的表现：** 使用 `--scope campus` 或 `--scope intern` 时返回"请求成功"但实际没有数据。

**现在的表现：** 已修复筛选逻辑，正常返回平安对应类型的岗位。

---

### 米哈游校招搜索返回空

**之前的表现：** `mihoyo search --scope campus` 返回 0 个岗位。

**现在的表现：** 已修复渠道 ID 映射，可正常获取米哈游校招岗位（目前约 16 个在招岗位）。

---

### OPPO 校招岗位显示异常

**之前的表现：** `oppo search --scope campus` 返回 0 个岗位。

**现在的表现：** 修复了岗位类型标签识别，现在能正常返回 OPPO 应届生岗位。

---

### 零一万物不再显示校招

**之前的表现：** `zerooneai search --scope campus` 返回空，但没有明确提示。

**现在的表现：** 零一万物目前只有社招（没有校招渠道），使用 `--scope campus` 时会直接提示"该公司不支持校招"，不再静默返回空结果。
